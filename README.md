# 服务架构
一台境内服务器+一台境外服务器  
境内服务器：  
1.负责用户权限信息，账号信息的管理  
2.前端页面管理（通过nginx）  

境外服务器：  
1.负责接收并相应境内服务器发送的问题请求（通过基于http协议的open-feign微服务组件）
2.统计提问信息提问次数等
